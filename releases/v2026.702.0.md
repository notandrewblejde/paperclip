# Paperclip v2026.702.0

> Released: 2026-07-02

## Fixes

- **Routine variables persist on PATCH** - Explicit `variables` supplied to `PATCH /api/routines/{id}` are now persisted as provided, instead of being silently discarded when not referenced in the routine's title or description template. Shadow variables and metadata variables (e.g. `shadow`, `shadow_issue_id`) now survive the update round-trip. ([SPC-17557](/SPC/issues/SPC-17557))
